---
description: Execute commands using subagent-driven development approach
---

Use the superpowers:subagent-driven-development skill to execute any task or command.

This command transforms regular commands into subagent-driven execution:
- Creates fresh subagent for each independent task
- Provides code review between tasks for quality gates
- Enables fast iteration with systematic validation
- Maintains clear task separation and progress tracking

Perfect for complex multi-step tasks where each step benefits from focused attention and intermediate quality checks.
