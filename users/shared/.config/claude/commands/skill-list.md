---
description: Display list of available skills with brief descriptions
---

Show all available Claude Code skills with their names and descriptions.

This command provides a quick overview of all installed skills, making it easy to:
- See what capabilities are available
- Find the right skill for a task
- Get brief descriptions of what each skill does

**Usage:**
```
/skill-list
```

The skills will be displayed with:
- Skill name
- Brief description
- Usage context (if applicable)

**Integration points:**
- **Pairs with:** brainstorming (to choose appropriate skills), writing-skills (to understand available capabilities)
- **Required background:** Basic understanding of Claude Code skill system
