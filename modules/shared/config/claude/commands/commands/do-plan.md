# Do Plan - plan.md 기반 로컬 실행 및 구현

## Process

1. **환경 확인**
   - 로컬 개발 환경 설정
   - 필요한 의존성 확인

2. **작업 선택**
   - 다음 작업 항목 선택
   - 작업 요구사항 이해
   - 구현 계획 검토

3. **구현**
   - TDD 접근법 적용 (테스트 먼저 작성)
   - 코드 작성
   - 기존 코드 스타일 준수
   - 적절한 커밋 메시지 작성

4. **품질 보증**
   - 모든 테스트 통과 확인
   - 린트 실행 및 통과
   - 코드 리뷰 준비

5. **완료 처리**
   - todo.md에서 완료된 항목 체크
   - 진행 상황 기록

6. **다음 작업**
   - 완료된 작업 검토
   - 다음 우선순위 작업 확인
   - 필요시 계획 조정

## Important Notes
- 테스트가 실패하면 진행하지 않음
- 커밋은 자주, 의미있는 단위로
- 막히면 즉시 도움 요청

## Loop
- 모든 작업이 완료될 때까지 2-6단계 반복
