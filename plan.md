# ssulmeta-go 프로젝트 개발 계획

## 프로젝트 목표
테스트 코드가 실행 가능한 Go 프로젝트 구성

## 기술 선택
- **테스트 프레임워크**: Go 내장 `testing` 패키지
- **테스트 구조**: `*_test.go` 파일 패턴
- **테스트 도구**: `go test`, `go test -cover`, `go test -v`
